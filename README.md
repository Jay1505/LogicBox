# Logic Box - Project 2

## Description

Logic Box is a Python console application that provides two functionalities: generating star patterns and analyzing a range of numbers. The application uses an interactive menu-driven interface.

## Features

* Generate star (`*`) patterns
* Analyze numbers as even or odd
* Calculate the sum of a range of numbers
* Interactive menu with multiple options
* Exit functionality

## Technologies Used

* Python 3

## Concepts Demonstrated

* Loops (`for`, `while`)
* Conditional Statements (`if-elif-else`)
* Nested Loops
* User Input and Output
